### node.js 기반으로 개발 소스코드가 구성되어 있습니다.
test

1. git clone
2. 폴더 이동
3. 명령어 실행: npm i
4. 명령어 실행: 
    - npm run dev
    - npm run build 후 npm run start 로 서버 빌드
